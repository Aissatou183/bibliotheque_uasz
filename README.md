\# 📚 Bibliotheque en ligne UASZ



Projet réalisé en OCaml dans le cadre du Master 1 Informatique (GL)  

Université Assane Seck de Ziguinchor



\## 🎯 Objectif



Développer une bibliothèque en ligne permettant :

\- la gestion des utilisateurs

\- le partage de documents administratifs

\- la consultation et le téléchargement de fichiers



\## ⚙️ Technologies utilisées



\- OCaml

\- Dune

\- Dream (framework web)

\- SQLite (base de données)

\- Yojson (JSON)

\- Digestif (hash des mots de passe)



\## 🚀 Lancer le projet



```bash

dune build

dune exec bibliotheque\_uasz
